## 📦 Installation

### Prerequisites
- Node.js (Latest LTS Version)

### Run Locally

1. Clone the repository

```bash
git clone https://github.com/rohit21oc/saumya-birthday.git
```

2. Navigate to project directory

```bash
cd saumya-birthday
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```

5. Open browser

```bash
http://localhost:5173
```
